# 簡易加減乘除與二元一次方程式運算測驗與練習
本程式包含 加法、減法、乘法、除法 以及 二元一次方程式 之求解
為了確保難易度為簡單，所能選擇之數字範圍為整數，並不包含0，在除法及二元一次方程式求解的部分，亦有做相關設計，使其題目不會出現解為小數點後兩位以上之複雜數字。

功能包含：
- 加法題目產生、解答確認
- 減法題目產生、解答確認
- 乘法題目產生、解答確認
- 除法題目產生、解答確認
- 二元一次題目產生、解答確認
- 以鍵盤輸入之簡易UI
- 將測驗結果上傳至firebase
- 將測驗結至firebase抓取查詢

尚未完成功能：
- 圖形化之UI
- 二元一次題目並未支援產生負數係數之設定
- 除法並未支援以分數形式表示之解答
